# 📦 Feature: ขอเบิก / คืน Item Master

> ระบบจัดการ Item Master Inventory — Homepro & PJW Soft

---

## 1. การขอเบิก / คืน Item Master Inventory

### ขั้นตอนการใช้งาน

1. **เข้าสู่ระบบ Homepro** และไปที่โปรแกรม **Material Request**

2. **เลือกข้อมูลที่ต้องการเบิก**
   - เลือก **Plant**
   - เลือก **Machine**
   - เลือก **Job** ที่ต้องการเบิก Item Master

3. **คลิกเลือก Item Job**
   - ระบบจะแสดงหน้าต่างใหม่
   - เลือกประเภทรายการ: **ขอเบิก** หรือ **ขอคืน**
   - กดปุ่ม **บันทึก**

4. **รายการที่สร้างจะปรากฏใน PJW Soft (Windows Form)**
   - ระบบ: **Master Inventory**
   - แท็บ: **Master Inventory Online**

5. **ผู้มีหน้าที่จ่าย** สามารถเลือกรายการที่ปรากฏ แล้ว **Scan Serial No** ได้ทันที

---

### 🗑️ การทำลาย Master Item

> กรณีที่ต้องการทำลาย Master ให้ดำเนินการในหน้าเดียวกัน

| ขั้นตอน | รายละเอียด |
|--------|------------|
| 1 | กรอก **Serial No** ที่ต้องการทำลายในกรอบ **Destroyed** |
| 2 | คลิกปุ่ม **Destroyed** เพื่อยืนยันการทำลาย |

---

## 2. 🆕 การต่ออายุ Tag (New Feature)

### ขั้นตอนการต่ออายุ Tag

1. **ไปที่แท็บ: `MI Control`**

2. **Filter ข้อมูล** รายการที่ต้องการต่ออายุ

3. **เลือกรายการที่ต้องการต่ออายุ** อย่างน้อย **1 รายการ**

4. **คลิกปุ่ม "ต่ออายุ Item"**
   - ระบบจะแสดงหน้าต่างพร้อมรายการที่เลือกไว้
   - ผู้ใช้งานสามารถ **แก้ไขข้อมูล** ในกรอบสีของแต่ละรายการได้

5. **กำหนดรอบการต่ออายุทั้งหมด (Optional)**
   - กรอกข้อมูลในช่องด้านบน
   - ระบบจะอัปเดต **รอบการต่ออายุ (เดือน)** ของทุกรายการทันที

6. **เลือกประเภท Tag** และคลิกปุ่ม **"Save And Generate Tag"**
   - ระบบจะขอให้ยืนยันการ **Generate Tag**
   - ยืนยันการ **Print Tag**

---

## 3. 🆕 การ Print Tag (New Feature)

### ขั้นตอนการ Print Tag

1. **ไปที่แท็บ: `MI Control`**

2. **คลิกปุ่ม "Print Tag"**

3. **Filter ข้อมูล** ตามความต้องการ

4. **เลือกประเภทการพิมพ์**

   | ประเภท | เงื่อนไข |
   |--------|----------|
   | **Print ปกติ** (ยังไม่เคยพิมพ์) | **ไม่ต้อง** เลือก Checkbox Re-print |
   | **Re-print** (เคยพิมพ์แล้ว) | **ต้อง** ติ๊กเลือก Checkbox Re-print และระบุ **เหตุผล** |

5. **คลิกปุ่ม "Print Tag"** เพื่อพิมพ์

---

## 📌 สรุปแท็บที่ใช้งาน

| แท็บ | การทำงาน |
|------|----------|
| **Master Inventory Online** | ดูรายการขอเบิก/คืนที่สร้างจาก Homepro |
| **MI Control** | ต่ออายุ Tag / Print Tag |

> **หมายเหตุ:** ฟีเจอร์ **ต่ออายุ Tag** และ **Print Tag** เป็น New Feature ที่เพิ่มใหม่ในระบบ PJW Soft
